Lens Search

단축키 하나로 화면 캡처 → 즉시 Google Lens 검색

다운로드 (LensSearch.zip)
1
다운받은 zip 압축 해제
다운로드 폴더에서 LensSearch.zip을 더블클릭하면 폴더가 생겨요.
2
Chrome 확장 프로그램 페이지 열기
Chrome 주소창에 아래를 복사해서 붙여넣으세요:
chrome://extensions
그리고 우측 상단 "개발자 모드"를 켜세요.
3
확장 프로그램 설치
"압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭하고, 아까 압축 해제한 폴더를 선택하세요.
4
단축키 설정
Chrome 주소창에 아래를 붙여넣으세요:
chrome://extensions/shortcuts
Lens Search 옆에 Cmd+Shift+L을 입력하세요.
사용법: 아무 웹페이지에서 Cmd+Shift+L → 영역 드래그 → Google Lens 검색 결과가 새 탭에 열려요!
